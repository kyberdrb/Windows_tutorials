Windows 10 Fall Creators Update

Ak bol pocitac pouzivany, najprv vykonat v metro nastaveniach: Windows Update -> Recovery -> Reset this PC -> Keep my files.
Potom pokracovat vecami uvedenymi nizsie.

NASTAVENIE PREDVOLENEHO ADMINISTRATORSKEHO UCTU

  -aktivovat Windows
  -.Net Framework 3.5 ("Enable Windows Features" v ovladacom paneli v "Programs and Features") - kvoli niektorym aplikaciam napr. MS Office
  -nainstalovat vsetky potrebne ovladace (ak je vyzadovany restart, urobime tak okamzite)
  -vypnut defragmentaciu (Win 10 si stale mysli, ze SSDcka potrebuju defragmentovat)
  -nechat nainstalovat vsetky dostupne aktualizacie - niektore aktualizacie mozu zmenit nastavenia Windowsu, preto budeme menit nastavenia Windowsu po nainstalovani vsetkych dosial dostupnych aktualizacii. Takisto sa automaticky nainstaluju aj najnovsie ovladace pre potrebne zariadenia z Windows Update (toto coskoro vypneme). Restartujeme pocitac, akonahle si to Windows Update vyzaduje.
  
  -vypneme zvukove efekty vo Windowse:
    -pravy klik na ikonku reproduktora (speaker icon) na start liste (taskbar) -> Zvuky (Sounds) -> "Zvukovu schemu" (Sound scheme) zmenime na "Bez zvukov" (No Sounds)

  -Otvorime "Kos" na pracovnej ploche -> v navigacnom paneli klikneme pravym na "Tento pocitac" (This PC) -> Vlastnosti (Properties) -> Pokrocile nastavenia (Advanced system settings)
    -"Advanced" karta
      -v casti "Vykon" (Performance) -> kliknut na tlacitko "Nastavenia" (Settings)
        -"Data Execution Prevention" karta
          -zapnut pre vsetky programy okrem nizsie uvedenych (zvysenie bezpecnosti pocitaca)
    -"System Protection" karta
      -zapnut iba pre systemovy disk (pre vsetky ostatne disky nechame vypnute) a okamzite vytvorit bod obnovy

  -Zmenit "Moznosti priecinka" (Folder options) -> Otvorime napr. Kos (Recycle Bin), v hornej liste klikneme na "Zobrazit" (View) a potom na "Moznosti" (Options):
    -karta "Vseobecne" (General)
      -Open File Explorer to: "This PC"
      -v casti "Privacy":
        -odskrtnut vsetky checkboxy (vypneme Quick access - Rychly pristup)
        -klikneme na tlacitko "Clear" (Vymazat)
    -karta "Zobrazenie" (View)
      -zaskrtnut "" (Display the full path in the title bar)
      -odskrtnut "Skryvanie znamych pripon suborov" (Hide extensions for known file types)
      -odskrtnut "Skryvanie konfliktov pri spajani priecinkov" (Hide folder merge conflicts)
      -zaskrtnut "Otvorit kazde okno v samostatnom procese" (Launch folder windows in separate process)
      -odskrtnut "" (Show sync provider notifications)
      -pod castou "Navigacna lista" (Navigation pane):
        -zaskrtnut "Zobrazit vsetky priecinky" (Show all folders) - zobrazi Ovladaci panel aj Kos v navigacnej liste; niektore polozky / adresare budu zobrazene duplicitne (to osetrime neskor)
    -karta "Hladanie" (Search):
      -odskrtnut vsetko
      
    
  -Pravy klik na tento pocitac -> Vlastnosti -> Pokrocile nastavenia (Advanced settings)
    -"Advanced" karta
      -v casti "Performance" -> kliknut na tlacitko "Settings"
        -"Visual effects" karta
          -zvolit "Adjust for best performance"
          -mozeme zapnut tieto nastavenia
            -ukazat nahlady namiesto ikon
            -[VOLITELNE] zobrazit obsah okna pri presuvani
            -[VOLITELNE] zobrazovat priesvitny obdlznik vyberu
            -[VOLITELNE] vyhladit hrany pisma
        -"Advanced" karta
          -v casti "Processor scheduling" -> Adjust best performance for = "Programs"
          -[VOLITELNE - ak mame >=16GB RAM] v casti "Virtual memory" -> Change
            -odskrtnut "Automatically manage paging file size for all drives"
            -V zozname diskov kliknut na kazdy z nich a pre kazdy zvolit "No paging file" a zakazdym kliknut na tlacitko "Set"
            -Ak mame menej ako 16GB pamate, nechame Windows, aby spravoval velkost strankovacieho suboru (checkbox "Automatically manage paging file size for all drives" zostane zaskrtnuty a nevykonavame ziadne zmeny ALEBO to nastavime iba na jeden disk, pre vsetky ostatne paging file vypneme, a pre ten jediny disk ho nastavime na Custom size: 4096. Ak to nepomoze sekaniu, mozeme presunut paging file na iny disk)
        -"Data Execution Prevention" karta
          -zapnut pre vsetky programy okrem nizsie uvedenych
      -v casti "Startup and Recovery" -> kliknut na tlacitko "Settings"
        -odskrtnut "Time to display list of operating systems" -> zrychli sa bootovanie
    -"System Protection" karta
      -zapnut iba pre systemovy disk a okamzite vytvorit bod obnovy
    -"Remote" karta
      -odskrtnut "Allow Remote Assistance connections to this computer"
      -zvolit "Don't allow remote connections to this computer"
    -Pri potvrdeni nastaveni (tlacidlo "OK") pri vyzve na restart zvolime "Restartujeme neskor" (Restart later)

  -Do start menu napisat "regedit" -> pravy klik -> spustit to ako spravca
    -upravime nastavenia "Multimedia Class Scheduler"
      -pojdeme do "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile"

      -pojdeme do "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile\Tasks\Games"
        -vnutri vytvorime novu DWORD hodnotu (ak este nie je vytvorena) s nazvom "GPU Priority" a priradime jej hexadecimalnu hodnotu 8
