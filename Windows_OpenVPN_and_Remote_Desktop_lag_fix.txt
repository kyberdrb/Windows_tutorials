Powershell in user mode
check MTU sizes

    netsh interface ipv4 show subinterfaces

Powershell in admin mode
change MTU sizes on the OpenVPN interface

	netsh interface ipv4 set subinterface "OpenVPN Wintun" mtu=1500 store=persistent
